World of Tank 모작

개발 기간 : 2023.01 ~ 2023.02 (1개월)
개발 인원 : 2인 팀 프로젝트
장르 : Shooting

버전 관리 도구 : github
사용 라이브러리 : WinAPI, DirectX, ImGui, FMOD
주요 개발 언어 : C++

WOT 모작 설명 영상 링크

개발 목적 및 배움

월드 오브 탱크 게임을 모작하는 것으로
WinAPI, DirectX, ImGui, FMOD에 대한 이해도와
경험을 쌓으며 github를 이용한 팀프로젝트를
통하여 다른 사람과의 공동 개발을 경험하고
버전 관리 도구를 경험하는 것을 목표로
개발하였습니다.

싱글톤 방식의 디자인 패턴을 사용하여
팀원과의 개발 구역을 분리하고 데이터를
관리할 수 있었습니다.

Unreal 액션게임

개발 기간 : 2023.07 ~ 2023.09 (2개월)
개발 인원 : 1명 프로젝트
장르 : 액션 RPG

사용 엔진 : Unreal / Unreal C++

Unreal 액션 게임 모작 설명 영상 링크

개발 목적 및 배움

Unreal / Unreal C++를 이용한 게임 제작을 경험하고 Unreal에 대한 이해도 및 경험을 쌓는 것을 목표로 개발하였습니다.

아이템의 상호작용 및 스킬 구현이라는
목표를 정하고 이를 어떻게 구현할지를 고민하고
설계하는 것을 목표로 개발하였습니다.

객체지향 프로그래밍 원칙에 중심을 두어
개발하였으며 이를 통해서 개발의 확장과 유지보수를 용이하게 할 수 있었습니다.

포트폴리오 웹 페이지

개발 기간 : 2024.05 ~ 2024.05
개발 인원 : 1명
기능 : 간단한 자기소개, 인적 사항 및 프로젝트 경험 소개


사용 언어 : HTML, CSS, javascript

개발 목적 및 배움

마크업 언어 HTML,CSS와 스크립트 언어 JavaScript를 통해서 다양한 언어를 활용하고 사용하는 능력을 향상시키는 것을 목표로 학습 및 개발하였습니다.

웹 페이지 개발을 통해서 UI 개발 경험 및 능력을
향상시키고 보다 좋은 UX를 제공하는 방법과
디자인을 고민하고 설계하는 것을
목표로 개발하였습니다.

차량 찾기 프로그램

개발 기간 : 2024.06 ~ 2024.06
개발 인원 : 1명
기능 : 영상으로 부터 차량 검색 및 영상 저장

사용 언어 : C++
사용 라이브러리 : opencv 4.6.0
사용 딥러닝 : YOLOv3

객체 찾기 프로그램 영상

개발 목적 및 배움

이미지 처리 라이브러리 opencv와 딥러닝 YOLOv3를 통해서 영상 처리 기법을 공부하고 딥러닝과 머신러닝에 대해서 공부하고 다루는 것을 목표로 학습 및 개발하였습니다.

이미지를 처리하고 영상을 다루며 이를 저장하는 라이브러리인 opencv 를 통해서 영상 및 이미지 처리 능력을 향상시키고 YOLO를 공부하고 다루며
딥러닝에 대한 정보를 배우고 이를 학습하고 다루는 능력을 기를 수 있었습니다.
